# Card News Design System

인스타그램 카드뉴스 아카이브용 디자인 시스템. 모든 토큰은 `shared/styles.css`의 CSS 변수로 정의되어 있어 이 파일만 수정하면 전체 템플릿에 반영됨.

---

## 1. Canvas

| 항목 | 값 |
|---|---|
| 가로 × 세로 | **1080 × 1350 px** (4:5) |
| 안전 여백 (safe area) | 상하 120px / 좌우 80px |
| 최대 콘텐츠 너비 | 920px |

> 캡처 방법: 브라우저에서 HTML 열기 → DevTools → 디바이스 1080×1350 설정 → Full-size screenshot.

## 2. Typography

한글: **Pretendard** (오픈소스, 카드뉴스 표준)
숫자/영문: **Inter** (보조, 숫자 비율 좋음)

CDN으로 로드 — 별도 설치 불필요.

### 타이포 스케일

| 역할 | 크기 | 굵기 | 행간 | 용도 |
|---|---|---|---|---|
| Display | 96px | 800 | 1.15 | 표지 메인 타이틀 |
| H1 | 72px | 700 | 1.2 | 각 장 대표 문장 |
| H2 | 56px | 700 | 1.25 | 소제목 |
| Body-L | 40px | 500 | 1.45 | 본문 강조 |
| Body | 34px | 400 | 1.5 | 본문 기본 |
| Caption | 26px | 500 | 1.4 | 보조 설명 |
| Label | 22px | 600 | 1.2 | 카테고리/태그 (대문자 tracking 0.1em) |

숫자는 `font-feature-settings: "tnum"` 적용해 세로 정렬 맞춤.

## 3. Color

**미니멀 원칙**: 무채색 5단계 + 포인트 컬러 1개.

### Neutral

| 이름 | HEX | 용도 |
|---|---|---|
| `--ink` | `#111111` | 메인 텍스트 |
| `--ink-soft` | `#444444` | 서브 텍스트 |
| `--ink-muted` | `#888888` | 메타 정보, 캡션 |
| `--line` | `#E5E5E5` | 구분선, 테두리 |
| `--surface` | `#F5F5F5` | 카드 내 섹션 배경 |
| `--bg` | `#FFFFFF` | 캔버스 배경 |

### Accent (포인트 1컬러)

| 이름 | HEX | 설명 |
|---|---|---|
| `--accent` | `#0066FF` | 기본값: 선명한 블루. 지적이고 중립적 |
| `--accent-soft` | `#E6F0FF` | 포인트 배경 (하이라이트, 칩) |

> **쉽게 바꾸기**: `shared/styles.css`의 `--accent` 두 줄만 수정.
> 추천 대체 컬러: `#FF5A1F` (비비드 오렌지) / `#E03131` (레드) / `#1D7F5E` (딥 그린) / `#6B4EFF` (퍼플)

### 사용 규칙
- **배경**: 항상 `--bg` (흰색). 섹션 강조 시에만 `--surface`.
- **포인트 컬러**: 한 장당 최대 2곳 (숫자, 하이라이트 바, 아이콘). 남발 금지.
- **차트/데이터**: 메인은 `--accent`, 비교군은 `--ink`.

## 5. Layout Anatomy

모든 템플릿 공통 구조:

```
┌─────────────────────────────────┐
│  [top-meta]  Label · page 1/7   │ ← 22px muted
│                                  │
│                                  │
│  [main content]                  │ ← 가변 영역
│                                  │
│                                  │
│                                  │
│  [bottom] brand / footer         │ ← 22px muted
└─────────────────────────────────┘
```

- **top-meta**: 좌측 = 템플릿 유형 라벨(`개요`, `비유` 등), 우측 = 페이지 번호
- **bottom**: 좌측 = 브랜드/핸들, 우측 = 출처/해시태그
- **main**: 템플릿별 자유

## 6. 14 Template Types

| # | 유형 | 고정 위치 | 핵심 요소 |
|---|---|---|---|
| 01 | 개요 | **첫 페이지** | 라벨 · Display 타이틀 · 한 문장 설명 |
| 02 | 비유 | 중간 | "마치 ___ 처럼" 구조, 좌/우 대응 |
| 03 | 오해와진실 | 중간 | X(오해) / O(진실) 상하 대비 |
| 04 | 단계 | 중간 | 01 → 02 → 03 번호 세로 리스트 |
| 05 | 타임라인 | 중간 | 좌측 세로선 + 시점·사건 |
| 06 | 비교 | 중간 | A vs B 2컬럼 표 |
| 07 | 장단점 | 중간 | Pros ↑ / Cons ↓ 상하 분할 |
| 08 | 스펙트럼 | 중간 | 수평 바 + 위치 마커 |
| 09 | 실생활사례 | 중간 | 이모지/아이콘 + 상황 카드 2~3개 |
| 10 | 통계숫자 | 중간 | 거대한 숫자 + 출처 |
| 11 | 결정트리 | 중간 | 질문 → Yes/No 분기 다이어그램 |
| 12 | 체크리스트 | 중간 | `☐` 리스트 |
| 13 | FAQ | 중간 | Q/A 2~3세트 |
| 14 | 한줄요약 | **마지막** | 중앙 큰 인용 + 추천 다음 행동 |

### 캐러셀 조합 규칙
- **고정**: 1장 = `개요`, 마지막 = `한줄요약`
- **가변**: 중간 3~5장을 02~13 중 개념 성격에 맞게 선택
- **총 길이**: 5~7장 권장 (표지 + 중간 3~5 + 마무리)

## 7. Accessibility & Readability

- 최소 폰트 크기: **22px** (Caption) — 모바일에서 읽힘 보장
- 본문 대비: `--ink` on `--bg` → 20:1 (WCAG AAA)
- 포인트 `--accent` on `--bg` → 4.6:1 (AA 통과)
- 한 장 최대 글자 수: **약 120자** (한글 기준) — 넘으면 분할

## 8. File Structure

```
card-news-templates/
├── design.md              ← 이 문서
├── shared/
│   └── styles.css         ← 디자인 토큰 + 공통 레이아웃
└── templates/
    ├── 01-overview.html
    ├── 02-analogy.html
    ├── ...
    └── 14-oneline.html
```

각 템플릿은 단일 HTML 파일. 더미 텍스트가 들어있어 내용만 교체하면 즉시 사용 가능.
